Create an argument builder with default settings.

Provided Methods§

source

fn args_builder_of_val(&self) -> Self::ArgsBuilder

Create an argument builder with default settings. Types of the values specified in #[bin_data(encode = ...)] is unknown to the bin_data macro, so we use this method to let the compiler infer a proper Self type and Self::ArgsBuilder.
